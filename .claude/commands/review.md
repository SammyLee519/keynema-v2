## 리뷰 기준

### 🔴 높음 (반드시 수정)
- TypeScript: any 사용, 타입 단언(!. as) 남용
- null/undefined 체크 누락으로 런타임 에러 가능성
- useEffect 의존성 배열 누락 또는 과다 포함
- Supabase 쿼리 에러 핸들링 누락

### 🟡 중간 (개선 권장)
- TanStack Query queryKey가 일관되지 않음
- 커스텀 훅으로 분리할 수 있는 로직이 컴포넌트 내부에 있음
- Props drilling이 2단계 이상인 경우
- 재사용 가능한 컴포넌트를 인라인으로 반복 작성

### 🟢 낮음 (선택적 개선)
- 변수/함수 네이밍이 의미를 충분히 전달하지 못함
- 컴포넌트가 너무 많은 역할을 담당 (SRP 위반)
- 접근성 (aria, 시맨틱 태그) 미흡
